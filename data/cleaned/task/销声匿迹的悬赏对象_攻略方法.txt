# 销声匿迹的悬赏对象_攻略方法

任务名称：销声匿迹的悬赏对象
任务类型：世界任务
任务区域：挪德卡莱
版本号：月之二

=== 攻略方法 ===

【】
触发条件：冒险家协会-情报交易中，使用【已记载的情报】解锁任务
等级限制：暂无
特殊限制：暂无

【任务概述】
前置任务：无
起始npc：雅珂达
结束npc：雅珂达
后续任务：无

【任务过程】
「旗舰」的悬赏榜单上有一个目标在犯下许多骇人罪行后便销声匿迹，仿佛人间蒸发。而罪咎终须追索，罪责终须清偿。询问「旗舰」员工勒塞克，或许能得到更多信息。
·前往「秘闻馆」
·与「旗舰」员工勒塞克对话

【剧情对话】
前往「秘闻馆」
已全部解决「旗舰」的三项悬赏
雅珂达 : 「旗舰」的悬赏啊…不知不觉就只剩最后一条了。
奈芙尔 : 听闻是多亏了旅行者的活跃，不过看来就算是他(她)，对这最后一条悬赏也是束手无策。
雅珂达 : 毕竟实在没什么线索嘛——
雅珂达 : 啊，悬赏上没写，倒也不见得真的没有。
奈芙尔 : 看来你有头绪了？
雅珂达 : 我去找勒塞克问问好了，凭我的资深身份，应该能问出些什么。
奈芙尔 : 那就交给你了。
雅珂达 : 嗯，放心吧，老板！
与「旗舰」员工勒塞克对话
勒塞克 : 悬赏目标的信息都在板上，问我也行，但赏金多少没有商量的余地。
雅珂达 : 关于悬赏目标…
勒塞克 : 板上有四个目标，但只有三个有线索。被人指控欺诈而悬赏的…
雅珂达 : …停一下，停一下！敷衍那些新入行的宝藏猎人也就算了，对我也这样？你知道我要问的是哪个。
勒塞克 : 「秘闻馆」要插手吗？
雅珂达 : 那得看老板的意思。欸？怎么变成你问我了？
雅珂达 : 说起来，「裂颚」哈尔克到现在还是下落不明，难道不是因为你们给的线索写得太模糊了吗？你自己看看，连外貌特征都没有。
勒塞克 : 要是哪个宝藏猎人连这个都不知道，那肯定也不是哈尔克的对手，何必让他们浪费时间…浪费自己的性命。
雅珂达 : 好吧，那除了他下巴上那道骇人的疤痕，还有什么其他的特征？最好是其他人不知道的。
勒塞克 : 其他人不知道的…你是说身上的，比如说背上的印记之类？
雅珂达 : 对，就是这种，原来真有啊！不过你又是怎么知道的？
勒塞克 : 我不知道。有没有我也不知道。再说，就算真的有，你准备怎么查验？
雅珂达 : 这、这么说来也是…唉，真的就没什么独家消息？看在咱们这么熟的份上，就不能透露一点吗？
勒塞克 : 哈尔克在那夏镇暴露了本性之后逃窜去了伦波岛北部一带，在那里继续为非作歹。
雅珂达 : 这我都知道，之前多多少少听过。
雅珂达 : 但他最近不是没了消息吗？要是去了北边，皮拉米达城还好，但要是去了至冬堡…
雅珂达 : （冰天雪地的，我这胳膊能受得了吗…）
勒塞克 : 不会，他还在伦波岛，只是不知道他在哪里藏身。这条是独家消息。
雅珂达 : 欸，这你又是怎么确定的？
勒塞克 : 那就是商业机密了，价钱是另算的。
雅珂达 : 嘁，小气。唉，一个下巴有疤的壮汉，哪怕只是在伦波岛上，走十步也能碰到一个吧，这要怎么找啊。
勒塞克 : 那就看在我们这么熟的份上，给你一个建议，别光盯着下巴上的疤痕。
雅珂达 : …你是想说，他会设法把下巴上的疤痕遮住？
雅珂达 : 有道理…他会被称作「裂颚」就是因为那道标志性的伤疤，反过来说，如果没了这道疤痕，他就和其他壮汉没什么区别了。
雅珂达 : 这样一来，只要找刻意遮挡下巴的壮汉就好了。算是一条新线索，好吧，之后有进展了再来找你。
勒塞克 : 等你的好消息。
未全部解决「旗舰」的三项悬赏
雅珂达 : 「旗舰」的悬赏里有一条很棘手呢，几乎没什么线索。
雅珂达 : 前阵子还听说他在伦波岛北边伤人呢，现在却突然消失得无影无踪，一点消息也没有…
奈芙尔 : 「凡走过必留痕迹」，哪怕是抹除痕迹，也会留下新的印迹。
奈芙尔 : 再者，悬赏里没有明文写出，也不代表他们没有掌握任何信息。
雅珂达 : 说得也是，我去找勒塞克问问好了，凭我的资深身份，应该能问出些什么。
奈芙尔 : 那就交给你了。
雅珂达 : 嗯，放心吧，老板！
